# **RELEASES**

## **v0.5.0 - The Brain Update**
*   **Real-time Updates**: File Explorer now updates automatically when files are changed on the server.
*   **Session Persistence**: Terminal sessions are now persistent. Reloading the browser resumes your active shell session.
*   **Theming**: Introduced a new Settings app to change Accent Color, Background Color, and Wallpaper. Themes are saved locally.
*   **Features**:
    *   Right-click Context Menu on Desktop and File Explorer.
    *   Rename, Copy, Paste, Delete files/folders.
    *   Drag and Drop to Move files into folders.
    *   Global click handler to close menus.
*   **Bug Fixes**:
    *   Fixed Desktop background click issue.
    *   Fixed recursive delete error.
    *   Fixed crash when moving files to folders.
    *   Isolated App context menus from Desktop.
